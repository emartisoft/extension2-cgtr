# CGTR Modern

**commodore.gen.tr** forumunu modern bir arayüze dönüştüren Chrome eklentisidir.

## Özellikler

- Forum SMF 1.1.21 temasını modern kart tabanlı bir tasarıma dönüştürür
- 3D dönen logo animasyonu
- Kullanıcı girişi yapıldığında hoş geldin paneli ve gökkuşağı efekti
- "Son Mesajlar" bölümü avatar + içerik kart düzeni
- Konu sayfasındaki resim butonları siyah zeminli metin butonlarına dönüştürür
- Animasyonlu arka plan ızgarası
- Gereksiz öğeler (reklam, slider, showcase vb.) gizlenir

## Chrome'a Yükleme

1. Chrome'da `chrome://extensions` adresine gidin
2. Sağ üstten **Geliştirici Modu**'nu (Developer Mode) açın
3. **Paketlenmemiş Uzantı Yükle** (Load Unpacked) butonuna tıklayın
4. Bu klasörü seçin (`extension2-cgtr`)
5. Eklenti aktif hale gelecektir

Kullanmak için `www.commodore.gen.tr/forum/` adresini ziyaret edin.
